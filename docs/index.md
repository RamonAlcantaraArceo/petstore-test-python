# petstore-test-python

A structured Python test repository supporting **API**, **UI** (Selenium + POM),
**integration**, and **end-to-end** testing of the
[Swagger Petstore](https://petstore.swagger.io/) with a shared interface and
fluent assertions.

## What's Inside

| Layer | Technology |
|---|---|
| Language | Python 3.12 |
| Package manager | uv |
| Testing | pytest, pytest-asyncio, Allure Report 3 |
| Test data | Faker, factory_boy |
| Coverage | pytest-cov → Codecov |
| UI automation | Selenium + Page Object Model |
| API client | requests |
| Type checking | mypy |
| Linting | ruff + black |
| Pre-commit | ruff, black, mypy |
| CI/CD | GitHub Actions |
| Docs | MkDocs + Material |
